1.Linux


创建目录：mkdir 所创建的位置/目录名字
mkdir -p 递归创建
删除目录：rm -rf(切记别删除根目录）
创建文件：touch 文件名
访问名称：whoami
改登录用户：su 要登的用户名
移动，重命名：mv（移动：文件所在位置及文件名 目标位置）
编辑文件，查看文件：vi
复制文件：cp -rp 目标文件或者目录 目标位置
查看目录：pwd
查看：ls -a全部 -l详细信息 -d目录属性
查看IP地址：ip addr

2.搭建服务器
用centos 调用ssh
1.systemctl enable sshd
2.systemctl start sshd
3.vim /etc/ssh/sshd_config esc :set nu显示行号
4.38行去掉#后面没有yes就添加yes即可，PermitRootLogin yes
5.systemctl restart sshd

用mobaxterm chs连接密钥会话，连接centos服务器，
yum install -y wget && wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh ed8484bec
在会话框中输入以上代码，下载宝塔服务器，bt 14查看用户名和密码以及网址，点开内网网址，登录宝塔，下载以下配置
打开网站，添加自己写的网站，关闭宝塔防火墙，远程连接mysql（在会话框中输入

  1. grant all privileges on *.* to 'root'@'%' identified by 'root' with grant option;
  2. flush privileges;

）
，修改数据库root密码，在MySQL中建立连接（切记开启phpstudy），连接中主机ip改成服务器的，密码为root密码，
连接建立后即可在浏览器中访问自己的网站，切记用http，不然跨域！
